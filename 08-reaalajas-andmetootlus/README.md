# 08-reaalajas-andmetootlus

Selles kaustas on teema "Reaalajas andmetöötlus" materjalid.

Tutvume sündmuspõhise arhitektuuriga ja voogandmete (streaming) lähenemisega andmeanalüütikas.

- [Baastase](./baastase/README.md) keskendub sündmuste ja reaalajaandmetöötluse mõistmisele, lihtsa publish/subscribe simulatsiooni loomisele ning tulemuste salvestamisele tabelisse.
- [Edasijõudnud](./edasijoudnud/README.md) on selle teema kiirema tempo ja suurema iseseisvusega rada, kus tutvutakse Apache Kafka põhikontseptsioonidega ja tehakse transformatsioone voogandmetel.
