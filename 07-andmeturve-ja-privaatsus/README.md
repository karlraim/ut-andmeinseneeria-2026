# 07-andmeturve-ja-privaatsus

Selles kaustas on teema "Andmeturve ja privaatsus" materjalid.

Käsitleme ligipääse, rolle ja tundlike andmete käsitlemist. Eesmärk on, et projektis oleks olemas realistlik turvalahendus ja privaatsuse kontroll, tuginedes minimaalõiguste printsiibile.

- [Baastase](./baastase/README.md) keskendub PII mõiste mõistmisele, lihtsa rollipõhise ligipääsu rakendamisele ning saladuste ja ligipääsude turvalisele hoidmisele.
- [Edasijõudnud](./edasijoudnud/README.md) on selle teema kiirema tempo ja suurema iseseisvusega rada, kus kujundatakse rollid ja vaated andmete maskeerimisega ning analüüsitakse auditit toetavaid logisid.
