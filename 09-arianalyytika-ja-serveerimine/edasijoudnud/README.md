# Praktikum 9: Ärianalüütika ja andmete serveerimine (Edasijõudnud)

## Eesmärk

Ehitada esitlusvalmis demo koos mõõdikute definitsioonidega. Praktikumi lõpuks oskad põhjendada mõõdikute arvutust ja seostada selle andmekvaliteedi ning turbenõuetega.

## Õpiväljundid

Praktikumi lõpuks osaleja:

- Suudab ehitada esitlusvalmis demo koos mõõdikute definitsioonidega
- Oskab põhjendada mõõdikute arvutust ja metoodikat
- Seostab mõõdikud andmekvaliteedi ja turbenõuetega
- Oskab kujundada andmepõhist lugu vastavalt sihtgrupile

## Teemad

| Teema | Kirjeldus |
|-------|-----------|
| Mõõdikute disain | KPI-de defineerimine, arvutusmetoodika ja piirangud |
| Dashboard ehitamine | Superset vms tööriistaga näidikulaua loomine |
| Andmelugu | Narratiivi kujundamine sihtgrupi järgi |
| End-to-end seos | Mõõdikute seostamine kvaliteedi, turvalisuse ja pärinevusega |

## Eeldused

- Kogemus eelmiste praktikumide teemadega (andmemudelid, kvaliteet, turvalisus)
- Töötav andmebaas analüütiliste andmetega
- Põhiteadmised SQL-ist ja visualiseerimisest

## Uued mõisted

| Mõiste | Selgitus |
|--------|----------|
| **Superset** | Apache Superset — avatud lähtekoodiga BI ja visualiseerimise platvorm |
| **Mõõdiku definitsioon** | Täpne kirjeldus, kuidas KPI arvutatakse (valem, filtrid, granularity) |
| **Slice & Dice** | Andmete vaatamine erinevatest dimensioonidest ja tasemetest |
| **Drill-down** | Koondandmetest detailandmetesse liikumine |
| **Data Storytelling** | Andmete esitlemine narratiivina, mis toetab otsustamist ja tegevust |
