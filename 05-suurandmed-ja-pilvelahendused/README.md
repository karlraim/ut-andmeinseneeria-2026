# 05-suurandmed-ja-pilvelahendused

Selles kaustas on teema "Suurandmed ja pilvelahendused" materjalid.

Tutvume suurandmete põhimõtetega ja pilve kasutusloogikaga. Mis on suurandmed? Miks on tekkinud pilvtöötlus? Eesmärk on osata valida vajadustele vastav formaat ja töötlusviis.

- [Baastase](./baastase/README.md) keskendub Parquet formaadi ja avatud tabeliformaatide mõistmisele, hajussüsteemide põhitõdedele ning esmastele liigutustele Databricks keskkonnas.
- [Edasijõudnud](./edasijoudnud/README.md) on selle teema kiirema tempo ja suurema iseseisvusega rada, kus tehakse Spark DataFrame põhiseid transformatsioone ja tutvutakse kaasaegsete andmejärvedega (data lakehouse).
