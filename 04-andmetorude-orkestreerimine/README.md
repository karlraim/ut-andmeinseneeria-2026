# 04-andmetorude-orkestreerimine

Selles kaustas on teema "Andmetorude orkestreerimine" materjalid.

Teeme andmetorustiku automaatseks. Algajad lahendavad ajastuse lihtsate vahenditega, edasijõudnud kasutavad orkestreerijat ja sõltuvuste mudelit.

- [Baastase](./baastase/README.md) keskendub skripti käivituse automatiseerimisele CRON ajastusega, töö logide kogumisele ja sõltuvuste põhimõtete mõistmisele.
- [Edasijõudnud](./edasijoudnud/README.md) on selle teema kiirema tempo ja suurema iseseisvusega rada, kus luuakse Airflow DAG ja seotakse laadimised ning testid ühte juhitavasse töövoogu.
