# 03-andmete-integreerimine

Selles kaustas on teema "Andmete integreerimine" materjalid.

Õpime andmeid koguma failidest ja API-dest ning laadima neid andmebaasi. Uurime erinevaid kaasaegseid integratsiooni meetodeid ja võtame kasutusele enimlevinud transformatsioonitööriistad (dbt).

- [Baastase](./baastase/README.md) keskendub ühe allika (CSV, Parquet, API) andmete puhastamisele ja andmebaasi laadimisele. Õpime selgitama ETL etappe oma sõnadega.
- [Edasijõudnud](./edasijoudnud/README.md) on selle teema kiirema tempo ja suurema iseseisvusega rada, kus tehakse inkrementaalne andmete laadimine ja ehitatakse logitud, veahaldusega ETL protsess.
