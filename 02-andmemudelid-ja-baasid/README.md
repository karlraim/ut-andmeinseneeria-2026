# 02-andmemudelid-ja-baasid

Selles kaustas on teema "Andmemudelid ja baasid" materjalid.

Keskendume relatsioonilisele ja dimensionaalsele andmete modelleerimisele. Eesmärk on, et õppija oskab kavandada andmebaasi struktuuri nii, et see oleks ärianalüütika seisukohast võimalikult optimaalne.

- [Baastase](./baastase/README.md) keskendub lihtsa tähtskeemi (star schema) loomisele, faktitabeli ja dimensioonide mõistmisele ning SQL päringute kirjutamisele joinide ja agregatsioonidega.
- [Edasijõudnud](./edasijoudnud/README.md) on selle teema kiirema tempo ja suurema iseseisvusega rada, kus rakendatakse Kimballi metoodikat, SCD Type 2 muudatuste jälgimist ja päringu jõudluse analüüsi.
